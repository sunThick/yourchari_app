# yourchari_app
自転車を投稿するSNSです。
クローン後、flutter pub run build_runner buildを実行してください。

## 作成背景・目的
カスタムされたオリジナル自転車のプラットフォームを作成しました。
このアプリを通してもっと人々に自転車を楽しんで欲しいと思います。

## 機能一覧
- ユーザー認証
- ログイン・ログアウト
- アカウント削除
- 自転車投稿・削除
- 投稿通報機能
- 画像投稿
- いいね
- ユーザーミュート
- プロフィール編集

## 使用技術一覧
- flutter
- firebase
- riverpod
- cloud functions
